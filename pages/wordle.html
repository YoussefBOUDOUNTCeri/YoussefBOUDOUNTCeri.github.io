Projet Wordle en Java

VOIR CODE Code sur GitLab

Description du Projet

Ce projet Java de Wordle, conçu pour le français, inclut un système d'indices basé sur Word2vec, une interface avec grille de jeu et clavier virtuel, et une jouabilité fidèle aux règles classiques de Wordle.

Fonctionnalités Clés

  • Portail de sélection au lancement
  • Affichage éléments de jeu et système d'indices
  • Sauvegarde des informations de partie
  • Intégration de Word2vec pour les indices
